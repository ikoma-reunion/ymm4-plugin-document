詳解

線形補間アルゴリズムの実装例です。

このアルゴリズムは、最も単純な補間方法の一つで、隣接する2つのデータ点の間を直線で結びます。 Tapsは1であり、GetWeight(double)は距離に基づいて線形に重みを計算します。

// 線形補間アルゴリズムのインスタンスを作成
IInterpolationAlgorithm linear = new LinearInterpolationAlgorithm();
// データ点
Span<double> data = stackalloc double[] { 10.0, 20.0, 30.0, 40.0 };
// 位置1.5での値を補間
double interpolatedValue = Interpolation.GetValue(linear, 1.5, data);
Console.WriteLine($"Interpolated value at 1.5: {interpolatedValue}"); // 期待値: 25.0
