詳解

キャッシュされた画像ファイルソースを表すクラスです。 IImageFileSourceインターフェースを実装し、画像データをDirect2D1ビットマップとして提供します。

このクラスは内部的にBitmapResourceを使用してビットマップを管理し、 キャッシュメカニズムを通じて画像の再利用を促進します。

// このクラスは通常、ImageFileSourceFactoryによって内部的に作成されます。
// 直接インスタンス化する例は複雑になるため、ここでは省略します。
// ImageFileSourceFactory.CreateImageFileSource("path/to/image.png"); のような形で利用されます。
