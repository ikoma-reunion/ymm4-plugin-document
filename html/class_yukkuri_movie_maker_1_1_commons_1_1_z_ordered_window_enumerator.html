詳解

指定されたWPFウィンドウのコレクションを、画面上のZオーダー（手前から奥への順序）で列挙します。

このクラスは、Win32 APIの GetTopWindow および GetWindow (GW_HWNDNEXT) を使用してウィンドウのZオーダーを走査します。 コンストラクタで渡されたウィンドウのリストに含まれるウィンドウのみを返します。 IDisposableを実装していますが、現在の実装では解放するアンマネージドリソースはありません。

// 現在のアプリケーションで開かれているすべてのウィンドウをZオーダーで取得し、そのタイトルをログに出力する例
public void LogWindowsByZOrder()
{
var windows = Application.Current.Windows.OfType<Window>();
Console.WriteLine("Windows by Z-Order:");
using (var enumerator = new ZOrderedWindowEnumerator(windows))
{
while (enumerator.MoveNext())
{
Window currentWindow = enumerator.Current;
Console.WriteLine($"- {currentWindow.Title}");
}
}
}
