詳解

WMI (Windows Management Instrumentation) から情報を取得するためのユーティリティクラスです。

このクラスは、WMIクエリを実行し、指定されたプロパティの情報を辞書形式で返します。 システム情報やハードウェア情報の取得に利用できます。 ManagementObjectSearcherとManagementObjectCollectionを適切にDisposeすることで、 リソースリークを防ぎます。

// CPUの情報を取得する例
Console.WriteLine("--- CPU Information ---");
foreach (var info in WMIInfo.GetInfo("Win32_Processor", "Name", "NumberOfCores", "NumberOfLogicalProcessors"))
{
Console.WriteLine($"Name: {info.GetValueOrDefault("Name", "N/A")}");
Console.WriteLine($" Cores: {info.GetValueOrDefault("NumberOfCores", "N/A")}");
Console.WriteLine($" Logical Processors: {info.GetValueOrDefault("NumberOfLogicalProcessors", "N/A")}");
}
// OSの情報を取得する例
Console.WriteLine("\n--- OS Information ---");
foreach (var info in WMIInfo.GetInfo("Win32_OperatingSystem", "Caption", "OSArchitecture", "Version"))
{
Console.WriteLine($"Caption: {info.GetValueOrDefault("Caption", "N/A")}");
Console.WriteLine($" Architecture: {info.GetValueOrDefault("OSArchitecture", "N/A")}");
Console.WriteLine($" Version: {info.GetValueOrDefault("Version", "N/A")}");
}

引数

戻り値
プロパティ名と値のペアを含む辞書のコレクション。

このメソッドは、ManagementObjectSearcherを使用してWMIクエリを実行し、 結果として得られる各オブジェクトから指定されたプロパティの値を抽出します。 取得される値は、WMIのデータ型に応じて適切な.NETの型に変換されます。
