詳解

ファイル拡張子とそれに対応するFileTypeを管理するコレクションです。

このクラスは ObservableCollection<T> を継承しており、コレクションの変更通知をサポートします。 アプリケーションで扱うファイルの種別（動画、音声、画像など）を拡張子に基づいて判別する機能を提供します。
