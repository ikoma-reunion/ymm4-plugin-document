詳解

画像ファイルソースを表すクラスです。 IImageFileSourceインターフェースを実装し、Direct2D1ビットマップを直接扱います。

このクラスは、既存のDirect2D1ビットマップをラップし、IImageFileSourceとして機能させます。 主に、既にメモリ上にロードされているビットマップを画像ソースとして利用する場合に用いられます。
