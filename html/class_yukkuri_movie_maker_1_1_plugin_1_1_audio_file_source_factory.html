詳解

オーディオファイルソースを作成するためのファクトリクラスです。

// 例: "audio.wav"というパスとトラック0でオーディオファイルソースを作成
var audioSource = AudioFileSourceFactory.Create("audio.wav", 0);
if (audioSource != null)
{
Console.WriteLine($"Audio source created: {audioSource.GetType().Name}");
}
else
{
Console.WriteLine("Failed to create audio source.");
}
