詳解

個々のファイル拡張子とその関連付けられたファイルタイプの設定を表すクラスです。

このクラスは、アプリケーションが認識するファイルの種類（動画、音声、画像など）を ファイルの拡張子に基づいて定義するために使用されます。 ValidatableBindableを継承しており、データバインディングと入力検証をサポートします。
