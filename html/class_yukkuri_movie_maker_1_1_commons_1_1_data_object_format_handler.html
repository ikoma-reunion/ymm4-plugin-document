詳解

クリップボードやドラッグアンドドロップで渡される IDataObject を操作し、 ファイル、テキスト、YMM4独自のJSON形式などのデータを統一的に扱うためのヘルパークラスです。

このクラスは、クリップボードやD&D操作から取得したT:System.Windows.IDataObjectをラップし、 含まれるデータの種類を判別したり、データを抽出したりするためのメソッドを提供します。 YMM4独自のJSON形式（タイムラインアイテムやエフェクトなど）のシリアライズ・デシリアライズもサポートしており、 アプリケーション内でのデータ交換を容易にします。
