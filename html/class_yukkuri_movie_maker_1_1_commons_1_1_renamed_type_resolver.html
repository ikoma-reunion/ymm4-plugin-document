詳解

型の名前変更を解決するためのユーティリティクラスです。 以前のバージョンで名前が変更された型（特にエフェクト関連）を、現在の新しい名前にマッピングする機能を提供します。

アプリケーションのバージョンアップに伴い、クラスのnamespaceやアセンブリ名が変更されることがあります。 このクラスは、古い設定ファイルやプロジェクトファイルから読み込まれた型名が、現在のアプリケーションで認識できるように変換する役割を担います。 主に、YukkuriMovieMakerの組み込みエフェクトがコミュニティプラグインとして再編成された際の互換性維持のために使用されます。

// 旧バージョンでのエフェクトの完全修飾名
string oldEffectTypeName = "YukkuriMovieMaker.Project.Effects.BinarizationEffect, YukkuriMovieMaker";
// 新しい名前に解決
string newEffectTypeName = RenamedTypeResolver.ResolveTypeName(oldEffectTypeName);
Console.WriteLine($"Old Type Name: {oldEffectTypeName}");
Console.WriteLine($"Resolved Type Name: {newEffectTypeName}");
// 期待される出力:
// Old Type Name: YukkuriMovieMaker.Project.Effects.BinarizationEffect, YukkuriMovieMaker
// Resolved Type Name: YukkuriMovieMaker.Plugin.Community.Effect.Video.Binarization.BinarizationEffect, YukkuriMovieMaker.Plugin.Community
// 変更されていない型名の場合
string unchangedTypeName = "System.String, mscorlib";
string resolvedUnchangedTypeName = RenamedTypeResolver.ResolveTypeName(unchangedTypeName);
Console.WriteLine($"\nUnchanged Type Name: {unchangedTypeName}");
Console.WriteLine($"Resolved Unchanged Type Name: {resolvedUnchangedTypeName}");
// 期待される出力:
// Unchanged Type Name: System.String, mscorlib
// Resolved Unchanged Type Name: System.String, mscorlib
